API Documentation

This is the documentation for the Movie API.

API Endpoints

Request URL Responses
GET /movies Returns a list of all movies
GET /movies/:title Returns general data
GET /genres/:name Returns data about a genre
GET /directors/:name Returns data about director
POST /users/register Allows new users to register
PUT /users/:email Allows users to update their username
POST /users/favorites/:movieTitle Allows users to add a movie to their list of favorites
DELETE /users/favorites/:movieTitle Allows users to remove a movie from their list of favorites
DELETE /users/deregister/:email Allows existing users to deregister
